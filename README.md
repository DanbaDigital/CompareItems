# CompareItems

A World of Warcraft Vanilla addon for the Turtle WoW Private server that allows you to compare items in your bags to equipped items by holding the Shift key while hovering over them.

## Features

- Hold Shift while hovering over items in bags, bank, or other containers
- Displays a comparison tooltip showing the equipped item's stats side-by-side
- Works with all equippable items (weapons, armor, accessories)

## Installation

1. Download the addon files
2. Extract the `CompareItems` folder to your WoW AddOns directory:
   - For Turtle WoW: `Turtle WoW\Interface\AddOns\CompareItems`
3. Restart WoW or reload your UI with `/reload`
4. The addon will automatically load and work

## Usage

- Hover over any item in your bags
- Hold the Shift key
- A comparison tooltip will appear showing your currently equipped item in the same slot

## Compatibility

- Designed for WoW Vanilla (1.12.1)
- Tested on Turtle WoW Private server
- Should work with other Vanilla private servers

## Notes

- This addon replicates the item comparison feature from modern WoW that is missing in Vanilla
- No configuration options currently available
- If you encounter issues, ensure the addon is properly installed and enabled
